# Vendored SIF subset

This directory contains only the SIF problems used by the FATROP research
benchmarks:

- `DTOC1L.SIF`
- `DTOC3.SIF`
- `DTOC6.SIF`

The files, upstream README, and license were taken from
[`ralna/SIF`](https://github.com/ralna/SIF) commit
`ba729aec456b4a38f02917a305d8719b26b4c9fc`.

The complete SIF collection is intentionally not vendored because it is over
one gigabyte. Clone the upstream repository when additional test problems are
needed.
